sliderule-ex/sliderulex.py

*Outline
This is a simple test program in Python that generates practice problems for a tape measure. This program allows you to learn how to use a tape measure by yourself.

*Installation
Run the program in Python.

*Contents
　Console input and output.
　There are five questions.
　Each question is a chain of 2 to 4 calculations.
　The calculations randomly include multiplication, division, square, sin, cos, and tan.
　The problems are presented in reverse Polish notation. A number is displayed first, followed by an operator or a second number. Use  slide rule to calculate the value.
　When you enter the result in response to a prompt, the error rate is evaluated: "Excellent!" if the error is within 1% for a single calculation, and "Ooops! If the error is less than 1% for one calculation, "Excellent!
　You can quit the program by pressing Ctrl-C.

・概要
これは計算尺の練習問題を生成するPythonの簡単で試験的なプログラムです。このプログラムによって計算尺の操作を独習できます。

・インストール
Pythonで実行してください。

・内容
　コンソール入出力です。
　5問出題されます。
　各問は2～4個の計算が連鎖しています。
　計算には乗算、除算、開平、2乗、sin、cos、tanがランダムに含まれます。
　問題は逆ポーランド記法で表示されます。最初に数値が表示され、以後、演算子か二つめの数値が表示されます。計算尺で値を計算します。
　プロンプトに応じて結果を入力すると、誤差が評価されます。1回の計算について誤差が1%以内なら「Excellent！」、それ以外なら「Ooops！」と表示されます。後者の場合、問題を解いた回数にカウントされず、別問題でやり直しになります。
　Ctrl-Cで途中終了します。
